📌 Structura AI 

AI-powered infrastructure health monitoring platform that monitors bridge safety across India using satellite, seismic, weather, and traffic data — without physical sensors.

📖 About the Project

Structura AI Insights is an intelligent infrastructure monitoring system designed to analyze and ensure the safety of bridges using advanced technologies and data sources.

It leverages:

🛰️ Satellite & Seismic Data — Remote sensing for structural analysis

🌦️ Weather & Traffic Data — Monitoring environmental and load conditions

🤖 AI Crack Detection — Computer vision for detecting structural damage

🧬 Digital Twins — Virtual models of bridges for simulation and prediction

🌐 Multilingual Insights — Dashboard support for multiple Indian languages

🚀 Getting Started
🔧 Prerequisites

Make sure you have Node.js and npm installed.

⚙️ Installation
# Step 1: Clone the repository
git clone https://github.com/Saumiya11/Structura-AI.git

# Step 2: Navigate into the project folder
cd Structura-AI/structura-ai-insights-main

# Step 3: Install dependencies
npm install

# Step 4: Start the development server
npm run dev
